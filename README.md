# Journey Nest

A modern React-based tourism website for exploring beautiful destinations and discovering hotels. Built with a focus on responsive design and seamless user experience.

## Features

- **Explore Destinations:** Browse stunning travel locations with engaging visuals
- **Hotel Discovery:** View available hotels with essential traveler information
- **Responsive Design:** Optimized for mobile, tablet, and desktop
- **Modern UI:** Reusable React components with smooth animations and hover effects

## Tech Stack

- **React** – Frontend framework
- **Tailwind CSS** – Styling
- **React Icons** – Icon library
